_This project is in progress._

It is a university project for a [course](https://www.ece.ntua.gr/el/education/undergraduate/courses?courseId=3297&selectedFlow=core&semester=1) named Multimedia Technology. It uses Java and JavaFX.

The deliverable is **a filing system**, where the files have metadata such as:
- title
- category
- author
- version (starting from 1)
- creation date

and they consist of at least separate paragraphs.

There are 3 types of users:
- **simple users**, who have reading rights to the files assigned to them
- **authors**, who have additional writing rights to the files they authored
- **admins**, who have all other rights, and edit users and categories
